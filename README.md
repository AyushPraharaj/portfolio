# Ayush Kumawat — AI/ML Portfolio

Personal portfolio site for an AI/ML & Data Science learner. Built with React + Vite, animated with Framer Motion, and featuring a **live in-browser MNIST digit recognizer** powered by TensorFlow.js.

## Features

- **Modern AI aesthetic** — dark glassmorphism design with animated gradient orbs
- **Interactive ML demo** — draw any digit (0–9) on a canvas; a CNN predicts it live in your browser (no server, pure TF.js)
- **Smooth animations** — scroll-triggered fade/slide-up effects via Framer Motion
- **Fully responsive** — mobile-first layout
- **Sections** — Hero · About · Skills · Projects · Live Demo · Contact

## Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Generate the MNIST model (one-time, requires Python + tensorflow)
pip install tensorflow-cpu tensorflowjs
python scripts/generate_model.py

# 3. Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

> **Note:** The Live Demo section shows a "Model Not Found" message until you run `generate_model.py`. All other sections work immediately.

## Customizing

### Add / edit projects
Edit [`src/data/projects.js`](src/data/projects.js) — each entry takes `title`, `description`, `tech[]`, `github`, `demo`, `category`, `categoryColor`.

### Add / edit skills
Edit [`src/data/skills.js`](src/data/skills.js) — group skills by category with a `color` and `bg` for the chip style.

### Update personal info
- **Name / tagline** → `src/components/Hero.jsx`
- **Bio / learning list** → `src/components/About.jsx`
- **Social links** → `src/components/Contact.jsx`

---

## Deploying

```bash
npm run build        # outputs to dist/
```

Push to GitHub and connect to **Vercel** or **Netlify** — they auto-detect Vite and deploy on every push.

> Make sure to run `generate_model.py` and commit the `public/model/` files before deploying so the live demo works.
